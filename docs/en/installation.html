Installation

Install vtmate

Paste the next code in your terminal to download and install vtmate as a command .

This method of installation will auto-detect gpu acceleration and install the most suitable version available.

curl -fsSL https://vtmate.eu/install.sh | sh

If you prefer you can download it manually or compile it via cargo or manually from source code.

Install llm engine

Install ollama

curl -fsSL https://ollama.com/install.sh | sh

Optional: Install default llm model

ollama pull llama3.2:3b

Optional: OpenTTS support

docker pull synesthesiam/opentts:all
