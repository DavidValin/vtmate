Instalacion

Instalar con script de shell

Pega el siguiente codigo en tu terminal para descargar e instalar vtmate como un comando.

Este método de instalacion detecta la aceleración de gpu soportada e instalará la mejor versión disponible.

curl -fsSL https://vtmate.eu/install.sh | sh

Si lo prefieres puedes descargárlo manualmente o compilarlo via cargo o manualmente desde el código fuente.

Instalar motor de llm

Instalar ollama

curl -fsSL https://ollama.com/install.sh | sh

Opcional: Instalar el modelo LLM por defecto

ollama pull llama3.2:3b

Opcional: soporte para OpenTTS

docker pull synesthesiam/opentts:all
